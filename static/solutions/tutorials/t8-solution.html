Simple Exponential Smoothing (SES) is a time series technique used to smooth data and produce short-term forecasts. The central idea is to create a weighted average of past observations, where the weights decay exponentially as we move further into the past.

  • The smoothing parameter 𝛼 α determines how much weight is given to the most recent observation relative to the past smoothed values.

  • α = 0.2 → 20% weight on the current observation, 80% on the past. Observed a very smooth fitted line. Slow to react to recent changes, follows long-term trend. Short-term fluctuations are largely ignored, this is ideal for noisy data where you care about long-term trends.

  • α = 0.5 → Equal weight to the current observation and past smoothed value, hence a balanced response. Captures trend moderately, some responsiveness to recent changes. Produces a moderately responsive curve. Useful when you want a balance between smoothing and responsiveness

  • α = 0.8 → 80% weight on the current observation, only 20% on the past. Fitted line reacts quickly to recent fluctuations. More sensitive, less smooth. The fitted line responds quickly to changes But, it can overreact to noise. Best for data that changes rapidly or where recent observations are more important

Smaller α → smoother, ignores short-term changes. the estimate retains influence from many past observations.

Larger α → more responsive to recent changes, but may overreact to random fluctuations.

from statsmodels.tsa.holtwinters import SimpleExpSmoothing
from sklearn.metrics import mean_squared_error
import numpy as np

alphas = [0.2, 0.5, 0.8]
rmse_values = {}

for alpha in alphas:
    ses_model = SimpleExpSmoothing(oil).fit(smoothing_level=alpha, optimized=False)
    fitted = ses_model.fittedvalues
    rmse = np.sqrt(mean_squared_error(oil, fitted))
    rmse_values[alpha] = rmse

print(rmse_values)

Higher α gives lower RMSE because it reacts faster to recent changes in the series.

Lowest RMSE (α = 0.8) does not always produce the best forecast visually:

High α may overfit short-term fluctuations, making the forecast volatile.

Medium α may provide a smoother, more realistic forecast.
