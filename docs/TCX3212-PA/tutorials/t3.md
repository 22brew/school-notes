## Questions

**1.** In this exercise, we will use the **Obesity** (`obesity_data.csv`) dataset taken from [WHO GHO Data](https://apps.who.int/gho/data/view.main.CTRY2450A?lang=en). The dataset contains prevalence of obesity among adults for different countries divided by gender and year.

Data in this form (also known as the **wide form**) is not suitable for data analytics. In this tutorial, we will convert it into **long form** that is appropriate for analysis.

**(a)** Use `pd.read_csv()` to load the dataset into a variable named `obesity_data` and view the dataframe.

**(b)** Write code to extract the first row of `obesity_data` as a pandas Series of string type, remove missing values, and name it `years`.

**(c)** Extract the headers information from the fourth row as a pandas Series (of string type) and name it `headers`.

**(d)** Each numeric cell in the dataset contains 2 sets of values — the (average) percentage value and a range. We will work with just the first number. Define a function `remove_brackets()` that takes a string, removes the part containing `[XX.X – XX.X]`, and returns just the single percentage value.

Next, define another function `melt_func()` to convert a single row from `obesity_data` from wide form to return a new dataframe in long form with columns: `country`, `percentage`, `gender`, and `year`. Use `remove_brackets()` to extract numeric values, and map columns to `years` and `gender` using the `years` and `headers` Series.

> Each row in the original dataframe results in multiple rows — one row for each country, percentage, gender type, and year. Iterate through each column (2nd column onwards) and create a new row for each column.

**(f)** Apply `melt_func()` to all data rows in `obesity_data` to get the final form with all countries (`combined_df`).

**(g)** You might notice that the data types for the columns are wrong (e.g., `percentage` column should be numeric instead of character). Explicitly cast the datatype for each column to either character, category, or numeric.

**(h)** To get the final form (`final_df`), remove rows where there are missing `percentage` values and rows having `"Both sexes"` for gender.
