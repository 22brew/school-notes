## Questions

**1.** In this exercise, we will use the **Auto** (`Auto.csv`) dataset, which contains 397 different car observations. The variables are:

| Variable | Description |
|---|---|
| mpg | Miles per gallon |
| cylinders | Number of cylinders |
| displacement | Engine displacement (cubic inches) |
| horsepower | Horsepower of car |
| weight | Weight of car (pounds) |
| acceleration | Time to accelerate from 0 to 60 mph (secs) |
| year | Model year (1900s) |
| origin | Country of origin (1=American, 2=European, 3=Japanese) |
| name | Car name |

**(a)** There are some missing values in the CSV (denoted by `"?"`). Read the help pages for `read_csv` to see how to convert them to `NA` and ignore them in the analysis.

**(b)** Use the `statsmodels.formula.api` module to fit a simple linear regression using `mpg` as the response and `horsepower` as the predictor. Use `summary()` to print the results. Comment on the output.

**(c)** Write out the model in equation form. Is the relationship between the predictor and the response positive or negative? What does the coefficient for `horsepower` suggest?

**(d)** Comment on the quality of fit of the model.

**(e)** Is there a statistically significant association between the predictor and the response?

**(f)** What is the predicted `mpg` associated with a `horsepower` of 98? What are the associated 95% confidence and prediction intervals? Explain the conceptual difference between these two intervals. Which one would a consumer use to estimate the fuel economy of a single specific car? Why?

**(g)** Create a scatter plot of the response (y-axis) versus the predictor (x-axis) and overlay a regression line using seaborn. You may use matplotlib to customize the plot if needed.

**(h)** By looking at the scatter plot, the relationship appears nonlinear. Produce diagnostic plots for the regression fit. Do the residuals vs. fitted plot suggest that a linear fit is appropriate?

**(i)** In some cases, the true relationship between the response and the predictors may be nonlinear. One simple way is to use a transformed predictor. The data appear to have a quadratic pattern, suggesting a model of the form:

$$\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x + \hat{\beta}_2 x^2$$

To implement this in `statsmodels.formula.api`, apply a transformation to the predictor and wrap it within the `I()` function (e.g., `I(predictor**2)`). Generate a new model (`model2`) and interpret whether including the squared term improves model fit.

**(j)** Generate diagnostic plots for `model2` and examine the residuals vs. fitted plot. What difference do you see? How about the results from `summary()`?

**(k)** Fit a multiple linear regression model `model3` using `mpg` as the response and all other variables (except `name`) as predictors. Use `summary()` to print the results. Comment on the output.
