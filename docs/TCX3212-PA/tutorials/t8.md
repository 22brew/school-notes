## Questions

**1.** In this exercise, you will analyze and forecast annual oil production in Saudi Arabia using **Simple Exponential Smoothing (SES)**.

**(a)** Load `Oil.csv` and convert the `Year` column to datetime format. Set it as the index and create a pandas Series with yearly frequency. Plot the original time series. Describe any trends or patterns.

**(b)** Fit a Simple Exponential Smoothing (SES) model. Experiment with different smoothing levels (α = 0.2, 0.5, 0.8). Plot the fitted values for each α on the same graph. Discuss how changing α affects the smoothness of the series and responsiveness to recent trends.

**(c)** Using α = 0.9, generate a 5-year forecast.

**(d)** Calculate the RMSE between the fitted values and the actual data for each α. Which α gives the lowest error? Does the lowest RMSE always correspond to the "best" forecast visually? Explain.

---

**2.** In this exercise, you will load, visualize, and analyze a quarterly time series of **Australian tourist arrivals** from March 1999 – Dec 2015. Use the `Austourists.csv` dataset.

| Variable | Description |
|---|---|
| Date | Month of observation in DD/MM/YYYY format |
| Tourists | Number of tourists (in thousands) |

**(a)** Load `Austourists.csv`, convert the `Date` column to datetime. Set it as the index, ensure quarterly frequency, and plot the time series. Describe the observed trend and seasonal patterns.

**(b)** Decompose the training time series using both **additive** and **multiplicative** decomposition methods. Plot the trend, seasonal, and residual components for both. Compare the results and explain which decomposition better fits the data.

**(c)** Partition the data: use all data up to December 2014 as the training set and the last four quarters (2015) as the validation set. Explain why separating training and validation data is important.

**(d)** Using the training set, fit a **Holt-Winters (Triple Exponential Smoothing)** model with Additive Trend and Additive Seasonality. Apply the following fixed smoothing parameters:
- Smoothing Level (alpha): 0.3
- Smoothing Trend (beta): 0.1
- Smoothing Seasonal (gamma): 0.2

**(e)** Generate a forecast for the four quarters of 2015.

**(f)** Calculate the RMSE for your forecasts and explain what it tells you. Then calculate the MAPE for each quarter of 2015 and interpret the results.

**(g)** Plot the forecast and fitted training values against the actual validation data.
