## Questions

**1.** In this exercise, we will use the **Immigration** (`Immigration.csv`) dataset, which contains 2119 observations from a survey of UK residents examining attitudes toward immigration. The dependent variable, `better`, is a score from 0–10 based on the question: *"Is the UK made a worse or a better place to live by people coming to live here from other countries?"* (0 = Worse, 10 = Better).

| Variable | Description |
|---|---|
| better | Perceived impact of immigration (0–10 scale) |
| bornuk | Nativity status (0: Born in UK, 1: Foreign-born) |
| age | Respondent's age in years |
| sex | Respondent's gender (Male or Female) |
| educ | Educational level (categorical, 3 levels; 'tertiary' = higher education) |
| vote | Political party last voted for (Conservatives, Labour, other) |

**(a)** Some common ideas about immigration attitudes include that younger people tend to be more positive. Use `statsmodels.formula.api` to fit a simple linear regression using `better` as the response and `age` as the predictor. What does the coefficient estimate for `age` suggest?

**(b)** Add a second independent variable: `sex` (categorical: Male/Female). Update your regression model using the formula `better ~ age + sex`.

**(c)** Which category (Male or Female) is serving as the reference (baseline) group? How can you tell? Based on the coefficient for `sex[T.Male]`, do men or women have more positive attitudes toward immigration?

**(d)** Add a third independent variable, `educ` (categorical). What are its possible values? Based on the number of categories, how many dummy variables are needed?

**(e)** We want to compare all education levels to "Upper secondary". Update your regression model so that "Upper secondary" is the reference category for `educ` by using: `C(educ, Treatment(reference="Upper secondary"))` in your formula.

**(f)** How should you interpret the education coefficient estimates? Are they statistically significant? Explain the relationship between education and immigration attitudes in words.

---

**2. Interaction Terms**

An interaction term is used when the effect of one variable depends on the value of another. We want to see whether the effect of age on `better` differs for people with different political preferences (`vote`).

**(g)** Fit a regression model that includes `age`, `vote`, and an interaction term between them (`age:vote`).

> Hint: In Python, include an interaction between two variables A and B in your formula as `A:B`. You do not need to manually set a reference group for `vote`.

**(h)** Write the regression equation separately for each vote group (baseline, Labour, Conservative), showing how intercepts and slopes differ because of the interaction term.

**(i)** In words, explain: How does the effect of age on `better` change across political groups?

**(j)** Visualize the interaction using `sns.lmplot` to show the linear relationship between `better` and `age`. Use `hue='vote'` to display separate lines for each voting category.
