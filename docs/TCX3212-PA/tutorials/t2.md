## Questions

**1. Bird Flu Cases and Death Analysis**

In this exercise, download the **BirdFluCases** (`BirdFluCases.txt`) dataset from Tutorial/Lab 2.

**(a)** Compute the case counts of bird flu cases for each year.

**(b)** Similarly, compute the total number of bird flu deaths per year. Then, merge the results from (a) and (b) into a single DataFrame using an inner join, ensuring each row represents a unique year with both cases and deaths.

**(c)** Using `matplotlib.bar()`, create a stacked bar chart where each bar represents a year, with the bottom segment showing total cases and the top segment showing deaths. Discuss any potential misrepresentation or limitations of this visualization.

**(d)** Modify the chart to a clustered (side-by-side) bar chart to show deaths alongside cases, improving interpretability while correctly reflecting that deaths are a subset of cases.

---

**2.** In this exercise, we will still be using the **College** (`College.csv`) dataset from Tutorial/Lab 1.

**(a)** Use the `.describe()` function to produce a numerical summary of the variables in the dataset.

**(b)** Create a scatterplot matrix for the following variables using Matplotlib: `Apps`, `Accept`, `Enroll`, `Grad.Rate`.

**(c)** Generate the same scatterplot matrix as in (b) but differentiate between Private and Non-private universities using different colors.

**(d)** Use `matplotlib.boxplot()` to create side-by-side boxplots comparing the number of applications (`Apps`) for private and public colleges (`Private`). Label your axes and add a title.

**(e)** Produce a scatterplot of `Top10perc` (percentage of new students from the top 10% of their high school) against `Grad.Rate` (graduation rate). Use different colors to represent different percentages in `Top10perc` (≤30, ≤50, ≤70, >70).

**(f)** Use `matplotlib.hist()` to generate a histogram of `Top10perc`. Add the same color-coding scheme as in (e).

**(g)** The command `plt.subplots(2,2,figsize=(14,10))` allows you to divide the plotting window into multiple regions. Use this command to display the boxplot, scatterplot, and histogram from parts (d) to (f). How would you modify the layout to allow 3 plots in the same figure?

**(h)** Explore the dataset further using different chart types of your choice.
