## Overview

HDB Resale Price Prediction using machine learning. Group Papa project for TCX3212 Predictive Analytics.

**Team:** Group Papa
**Dataset:** HDB Resale Flat Prices (Jan 2017 onwards) from data.gov.sg

## Learning Objectives

- Translate business challenges into structured analytical objectives
- Execute the CRISP-DM workflow: Business Understanding, Data Understanding, Data Preparation, Modeling, and Evaluation
- Communicate technical findings to a professional executive audience

## CRISP-DM Framework

The project follows the first five phases of the CRISP-DM (Cross-Industry Standard Process for Data Mining) framework:

1. **Business Understanding** — Define specific business success criteria
2. **Data Understanding** — Audit data quality and explore the dataset
3. **Data Preparation** — Feature engineering, cleaning, imputation, encoding
4. **Modeling** — Compare at least two candidate algorithms with K-Fold Cross-Validation
5. **Evaluation** — Evaluate using performance metrics; justify final model selection

## Report Structure

1. **Project Context & Objectives** — Domain context and analytics objective
2. **Domain/Data** — Data sources, acquisition methods, feature taxonomies
3. **Methodological Framework** — CRISP-DM execution, technical hurdles, mitigation strategies
4. **Data Exploration** — Visualizations and descriptive analysis linked to modeling strategy
5. **Data Preparation** — Preprocessing pipeline (cleaning, imputation, encoding, feature selection)
6. **Modeling and Evaluation** — Candidate algorithms, validation, performance metrics, final model justification
7. **Discussion and Conclusion** — Predictions, conclusions, critique, and suggestions for improvement
