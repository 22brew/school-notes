from sklearn.metrics import mean_squared_error
import numpy as np

alphas = [0.2, 0.5, 0.8]
rmse_values = {}

for alpha in alphas:
    ses_model = SimpleExpSmoothing(oil).fit(smoothing_level=alpha, optimized=False)
    fitted = ses_model.fittedvalues
    rmse = np.sqrt(mean_squared_error(oil, fitted))
    rmse_values[alpha] = rmse
    print(f"α={alpha}, RMSE: {rmse:.2f}")

print("")

Why it is important to separate training and validation data:

  • Training data is used to fit the model parameters.
  • Validation data simulates unseen future data for fair evaluation.
  • This helps detect overfitting (good on training, poor on new data).
  • It supports objective model/parameter comparison before final forecasting.
