Model Equation:
mpg = 39.9359 + (-0.1578) horsepower

Interpretation of the horsepower coefficient:

  • The coefficient for horsepower is -0.1578.
  • This means that for every 1-unit increase in horsepower, the miles per gallon (mpg) decreases by approximately 0.1578 units.

Since the coefficient is negative, this indicates an inverse relationship:

  • cars with higher horsepower tend to have lower fuel efficiency (lower mpg).

model2 = smf.ols("mpg ~ horsepower + I(horsepower**2)", data=auto_data).fit()

# k) Multiple Linear Regression with all predictors (except name)
# Get all column names except 'name' and 'mpg'
predictors = [col for col in auto_data.columns if col not in ["mpg", "name"]]
formula = "mpg ~ " + " + ".join(predictors)

print(f"Formula: {formula}")
print()

model3 = smf.ols(formula, data=auto_data).fit()
print(model3.summary())
print()

print("\nKey Observations:")
print(
    f"1. R²: {model3.rsquared:.4f} - The model explains {model3.rsquared*100:.2f}% of variance in mpg"
)
print(f"2. Adjusted R²: {model3.rsquared_adj:.4f} - Accounts for number of predictors")
print()
print("3. Significant predictors (p < 0.05):")
for var in predictors:
    if var in model3.pvalues and model3.pvalues[var] < 0.05:
        print(f"   - {var}: coef={model3.params[var]:.4f}, p={model3.pvalues[var]:.4e}")
print()
print("4. Non-significant predictors (p >= 0.05):")
for var in predictors:
    if var in model3.pvalues and model3.pvalues[var] >= 0.05:
        print(f"   - {var}: coef={model3.params[var]:.4f}, p={model3.pvalues[var]:.4f}")
print()
print("5. The F-statistic tests whether at least one predictor is significant.")
print(f"   F-statistic: {model3.fvalue:.2f}, p-value: {model3.f_pvalue:.4e}")
print("   The model as a whole is statistically significant.")
