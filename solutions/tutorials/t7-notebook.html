d) Apply a 3rd-order and 12th-order centered moving averages, and plot the smoothed series overlaid on the original time series. Compare the results and explain how the window sizes affect smoothing.

In [50]:
# centered moving average with window size = 3 and 12
ma3 = ridership_ts.rolling(window=3, center=True).mean()
ma12 = ridership_ts.rolling(window=12, center=True).mean()

# Plot the original time series with the smoothed lines
fig, ax = plt.subplots(figsize=(8, 7))

ax = ridership_ts.plot(ax=ax, color="black", linewidth=0.25)
ma3.plot(ax=ax, color="blue", linewidth=0.5)
ma12.plot(ax=ax, color="red", linewidth=1, style="--")

ax.set_xlabel("Time")
ax.set_ylabel("Ridership")
ax.set_title("Original Time Series vs Centered Moving Averages")

ax.legend(["Ridership"])
plt.grid(True)
plt.show()

Comparison and Effect of Window Size:

  • The 3-month MA only lightly smooths the data but still follows the peaks and short-term fluctuations of ridership closely. It does not substantially remove the seasonality.
  • The 12-month MA smooths the data much more heavily, almost entirely removing the annual seasonal fluctuations (since the seasonal period length is 12) and revealing the underlying longer-term trend.
  • Conclusion: A larger window size causes more smoothing by averaging over a wider period, acting as a stronger filter that cancels out short-term variation and noise to highlight macro trends. A smaller window length leaves more volatility and seasonality in the series.

k) Implement a Python function regression_stats(actual, forecast) to evaluate forecast accuracy. The function should accept observed and forecasted time-series values as inputs and compute RMSE, MAE, MPE, and MAPE. Use this function to assess and compare different forecasting methods.

import numpy as np
import pandas as pd


def regression_stats(actual, forecast):
    """
    Compute forecast accuracy statistics:
    ME, RMSE, MAE, MPE, MAPE
    """
    actual = np.array(actual)
    forecast = np.array(forecast)

    errors = actual - forecast

    ME = np.mean(errors)

    RMSE = np.sqrt(np.mean(errors**2))

    MAE = np.mean(np.abs(errors))

    MPE = np.mean(errors / actual) * 100

    MAPE = np.mean(np.abs(errors / actual)) * 100

    # print results
    print("Regression Statistics:")
    print(f"Mean Error (ME): {ME:.4f}")
    print(f"Root Mean Square Error (RMSE): {RMSE:.4f}")
    print(f"Mean Absolute Error (MAE): {MAE:.4f}")
    print(f"Mean Percentage Error (MPE): {MPE:.4f}%")
    print(f"Mean Absolute Percentage Error (MAPE): {MAPE:.4f}%")
    print("")
